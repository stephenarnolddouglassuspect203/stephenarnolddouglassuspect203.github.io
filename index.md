## 🧱 Understanding the Changes
Windows manages many background tasks by default. These tasks keep the system stable for general office work, but they often restrict gaming performance. 

* Power Plans: Windows often throttles your parts to save energy. We unlock the maximum power state.
* Game DVR: Microsoft includes a, feature to record your last few minutes of play. This runs in the background at all times. Turning it off releases CPU and RAM resources.
* Standby RAM: Windows caches files in RAM for quick opening. This is good for programs, but it can crowd out active game data. We release this standby memory to ensure your game always has the space it needs.
